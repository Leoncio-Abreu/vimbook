### Função para barra de status 

         set statusline=%F%m%r%h%w\
            [FORMAT=%{&ff}]\
            [TYPE=%Y]\
            [ASCII=\%03.3b]\
            [HEX=\%02.2B]\
            [POS=%04l,%04v][%p%%]\ [LEN=%L]

Caso este código não funcione acesse [este
link](http://vim.wikia.com/wiki/Writing_a_valid_statusline)
